## Next.js

Next.js is the main framework for persona.fm. It is full stack framework, meaning that we do not need to use a separate backend framework. 

### React.js
React.js is a library for building front end applications. It is what Next.js is built on top of.

### Tailwind CSS
Tailwind CSS is a utility-first CSS library. It is used to style the components in the application with pre-defined and customizable classes.

### Typescript
A type-safe version of Javascript. It is used to write the code in the application.

## Supabase
AN open source platform that handles the database and authentication for the application.

### Google OAuth
A provider that integrates with Supabase to handle authentication.

### Drizzle ORM
Drizzle is used to handle the database operations in the application. It is a layer that allows us to interact with the database without writing raw SQL queries.

#### PostgreSQL
A database query language that Drizzle is built on top of.
